# v1.0 Roadmap Archive

**Archived:** 2026-04-12  
**Status:** shipped with deferred warning

## Milestone Summary

openOii v1.0 shipped the guided, resumable, creator-controlled core loop from raw story idea to final merged video. The shipped milestone includes durable LangGraph execution, approval-aware character/storyboard flow, workspace projection with realtime progress, selective rerun / creative control, and in-product final delivery.

## Key Accomplishments

- Shipped a durable LangGraph-backed end-to-end generation pipeline with same-thread resume and review-gate recovery.
- Established approval-aware character references, shot-bound cast, and storyboard review contracts that preserve identity into downstream video generation.
- Delivered a backend-authored, section-first infinite canvas workspace with realtime progress, placeholder states, and refresh-safe reconstruction.
- Added selective rerun and candidate/current promotion so creators can fix characters, shots, and clips without restarting the whole project.
- Completed final assembly and delivery inside the existing workspace with merge gating, stale-final handling, preview, download, and retry-merge UX.

## Deferred Warning

- **PROJ-02 provider portability** remains unproven as an end-to-end project-scoped flow. Global provider configuration exists, but v1.0 closure is accepted on the basis of core loop completion rather than per-project provider swapping.
