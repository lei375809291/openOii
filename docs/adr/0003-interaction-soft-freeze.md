# Interaction soft freeze during full UI redesign

Primary creator operations stay semantically stable: canvas **Selection Binding**, **Agent Column** conversation, **Cell** regenerate, generate/confirm/cancel, and **Workbench Shell** without document scroll. Soft freeze allows density, defaults (expanded panes, default inspector tab), shortened labels, and folding secondary entry points. Hard visual-only freeze was rejected as too rigid for tacit UX polish; reordering main paths (e.g. swapping Agent Column side or redefining binding) is out of scope for this program.
